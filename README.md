# XIVMine

This is the underlying repository for the [XIVMine website](https://xivmine.pages.dev/).
This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

If you wish to contribute, instructions are below. If you are not comfortable with editing content via GitHub, you can contribute and suggest feedback via our [Discord Server.](https://discord.gg/eByG8hraBM)

## How To Contribute

1. Fork and Clone the Repository
2. Setup Your Development Environment
3. Make and Test Your Changes
4. Create a Pull Request

### Fork and Clone the Repository
1. Fork the repository by clicking the **Fork** button in the top right corner of the page.
2. Open a terminal or command prompt and clone your forked repository by running:
```
git clone https://github.com/YOUR-USERNAME/REPOSITORY-NAME.git
```
Replace the URL with the link to your forked repository.

### Setup Your Development Environment

1. Install [Node.js and npm.](https://nodejs.org/en) Verify that you have it installed by running:
```
node -v
npm -v
```
2. Install the dependencies of the project by running:
```
npm install
```
### Make and Test Your Changes

1. Make the changes on your Fork and start the local server by running:
```
npm run start
```
2. If the command does not open the website automatically for you, open your browser and go to ```http://localhost:3000```, where the website should be running locally on your machine with any changes you made. Most changes made are reflected live without needing to restart the server.

### Create a Pull Request

1. Commit and push your changes and create a pull request.
2. Ensure you add a clear title and description with any changes you've made in detail.
3. Submit the Pull Request for review. If your Pull Request is approved, your changes will appear on the website. The website will take around 5-10 minutes to update.
